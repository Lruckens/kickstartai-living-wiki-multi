# Gemeente Westerwoude

**Type:** Organization (municipality / client)

## Summary
The client municipality for Project Bakkie. The project delivers a citizen-facing Q&A chatbot answering questions about permits and waste collection, built over Gemeente Westerwoude's public web pages.

## Role in project
- Client and primary stakeholder.
- Public web pages serve as the data source for the [[rag-pipeline]].
