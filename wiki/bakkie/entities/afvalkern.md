# AfvalKern

**Type:** System (client / external)

## Summary
Collection-scheduling and bin-registration system used by [[gemeente-westerwoude]], supplied by [[omgevingssoft-bv]]. Read/write access is contractually exclusive to OmgevingsSoft. Required integration for the assistant's schedule lookups, bin requests, and grofvuil pickup bookings — but the necessary API endpoints do not yet exist.
