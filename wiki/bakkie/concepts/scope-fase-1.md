# Scope — Fase 1

**Concept:** Project scope and phasing

## Summary
Fase 1 focuses exclusively on the **waste-collection domain**. The assistant must answer top topics in plain Dutch and, critically, **act** rather than merely inform.

## Top topics (2024 KCC categorisation)
1. Pickup schedule questions (~31%)
2. Grofvuil pickup booking (~18%)
3. Bin requests — new / replacement / extra (~14%)
4. Complaints: overflowing containers, dumping, missed pickups (~12%)
5. Milieustraten openingstijden & accepted materials (~9%)
6. Other — verhuizing, large-household tariff, tuinafval, etc. (~16%)

## Acting requirements
- Book a grofvuil pickup (via [[afvalkern]])
- File a melding openbare ruimte (via [[mor]])
- Request a bin (via [[afvalkern]])
- 24/7 availability; graceful handoff to a human agent
- AVG-compliant interaction logging (see [[avg-dpia]])

## Deferred (later phases)
Parking permits, paspoort/rijbewijs appointments, participation processes. Out of scope for fase 1.
