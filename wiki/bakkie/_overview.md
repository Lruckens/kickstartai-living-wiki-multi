# Project Bakkie — Overview

**Status:** scoping
**Last updated:** 2026-06-16

## What this project is about
Project Bakkie is an engagement with **[[gemeente-westerwoude]]** to build a conversational citizen assistant ("chatbot") for the waste-collection domain on the gemeente website. Unlike a pure information bot, the assistant must be able to **act** — book grofvuil pickups, file MOR meldingen, and request bins — not merely inform. See [[scope-fase-1]].

The driver is volume: Westerwoude handles ~38,000 waste-related citizen contacts per year (62% by phone via KCC), roughly twice the per-capita load of comparable municipalities, attributed to a complex split-collection system.
