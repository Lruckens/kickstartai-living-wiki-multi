# Project Bakkie — Overview

**Status:** prototype
**Last updated:** 2026-06-16

## What this project is about
Project Bakkie builds a citizen-facing Q&A chatbot for [[gemeente-westerwoude]], answering questions about permits and waste collection. The technical approach is a [[rag-pipeline]] over the municipality's public web pages.

## Current status
Scoping complete; moving into prototype phase following the [[2026-06-16-kickoff]] meeting. First milestone is a working prototype by 2026-07-01.
