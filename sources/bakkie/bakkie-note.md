# Kickoff note — Gemeente Westerwoude chatbot (Bakkie)

Date: 2026-06-16. Attendees: project lead Sanne Wielinga, municipality contact Joost de Vries.

Decision: the Bakkie project will build a citizen-facing Q&A chatbot for Gemeente Westerwoude,
answering questions about permits and waste collection. Tech: a RAG pipeline over the
municipality's public web pages. First milestone: a working prototype by 2026-07-01.
